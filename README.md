# Nexus Ai: The Future of Interactive Learning

Nexus Ai is a premium, AI-powered interactive book platform designed for senior-level engineering insights. Built with Next.js 15, FastAPI, and Supabase, it features real-time search, global AI translation, and a sleek, responsive reading experience.

## 🚀 Live Demo
**Frontend**: [https://nexus-ai-book.vercel.app](https://nexus-ai-book.vercel.app)
**Backend API**: [https://nexus-ai-api.vercel.app](https://nexus-ai-api.vercel.app)

## ✨ Features
- **Global AI Translation**: Read chapters in Urdu, Arabic, Spanish, and more.
- **Deep Search**: Instant indexed search across titles, headings, and markdown content.
- **Nested Topics**: Deep-dive into chapters with an expandable, animated sidebar navigation.
- **Responsive Design**: 100% mobile-ready with a dark-mode first aesthetic.
- **Identity Avatars**: Personalized user experience with reading goals and custom avatars.

## 🛠️ Tech Stack
- **Frontend**: Next.js 15 (App Router), Tailwind CSS v4, Framer Motion.
- **Backend**: FastAPI (Python), Deep-Translator (Google API).
- **Database**: Supabase (PostgreSQL & Auth).
- **Deployment**: Vercel (Frontend & Serverless Backend).

## 📖 Getting Started

### Prerequisites
- Node.js 18+
- Python 3.10+
- Supabase account

### Installation
1. Clone the repo:
   ```bash
   git clone https://github.com/raoraza12/nexus-ai-book.git
   ```
2. Frontend Setup:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
3. Backend Setup:
   ```bash
   cd backend
   pip install -r requirements.txt
   uvicorn app.main:app --reload
   ```
